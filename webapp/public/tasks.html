Tasks

Initial data integration and browsing

The raw particle data from the Dark Sky simulations has been made available as SDF files. In addition to the raw particle data, Halo Catalog data, and a third dataset linking the Halo Catalogs to create a Merger Tree database will be available. A framework to ingest, process, and organize this data should be created.

Halo identification and visualization

The particles that compose a halo are defined by mass, position, and velocity. The halo finding process takes these particles as input and produces an additional set of properties for the halo. Each halo is collapsed into a single row of data, with columns corresponding to each data point.

Over time, halos can move, merge, or split apart. Merger trees are used to describe the evolution of a halo at any given time. The goal of this task is to visualize the evolution of a halo over time.

Diving Deep into Halo Substructure

Provide a way to visualize and compare the substructure of individual halos. Characteristics such as mass, velocity, and acquisition time of particles should be comparable among substructure components.

Tying Everything Together

Merge the results of the tasks above into a single unified visualization tool that allows a user to explore the data from the Dark Sky simulations.
