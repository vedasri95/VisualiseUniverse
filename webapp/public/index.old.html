Introduction

Our understanding of the Universe we inhabit and how it has progressed in its 13.7 billion year lifespan is primarily driven by cosmological simulations. These simulations of the formation of structure in the Universe usually simulate dark matter, a type of matter that neither absorbs nor emits light, but is believed to account for most of the matter in the Universe. Its existence and properties are inferred from its gravitational interaction with visible matter.

These simulations require a large number of particles, on scales as large as 102403. Visualizations are desired for these particles and the structures they form through gravitational interactions over time. Structures are typically identified via "halo finding". Dark matter halos are components of a galaxy that are roughly spherical, extending from some main visible component of the galaxy. These halos can represent galaxies or clusters of galaxies. Substructures in these halos may be identified as smaller halos, representing galaxies. The interactions of these structures and substructures shape the structure of the Universe as a whole. Thus, visualizing the transitions these halos go through can provide great insight to the history and future of our Universe.

The goal of this project is to create a tool to visualize this data set.

Tasks

Initial data integration and browsing

The raw particle data from the Dark Sky simulations has been made available as SDF files. In addition to the raw particle data, Halo Catalog data, and a third dataset linking the Halo Catalogs to create a Merger Tree database will be available. A framework to ingest, process, and organize this data should be created.

Halo identification and visualization

The particles that compose a halo are defined by mass, position, and velocity. The halo finding process takes these particles as input and produces an additional set of properties for the halo. Each halo is collapsed into a single row of data, with columns corresponding to each data point.

Over time, halos can move, merge, or split apart. Merger trees are used to describe the evolution of a halo at any given time. The goal of this task is to visualize the evolution of a halo over time.

Diving Deep into Halo Substructure

Provide a way to visualize and compare the substructure of individual halos. Characteristics such as mass, velocity, and acquisition time of particles should be comparable among substructure components.

Tying Everything Together

Merge the results of the tasks above into a single unified visualization tool that allows a user to explore the data from the Dark Sky simulations.

Data

The raw particle data from the Dark Sky simulations is available in SDF files. Halo Catalog and Merger Tree data sets generated using Rockstar are also available, in tabular ASCII format. A relatively small (11GB) dataset generated by a 128^3 particle simulation has been provided for initial development.

Visualization methods will be developed and tested locally using this smaller dataset. The full raw particle data set is 31TB in size. After visualization methods have been developed, the Blue Waters supercomputer will be used to run these visualizations on a larger data set.

Software

The software developed will be primarily written in Python, utilizing libraries such as yt for SDF file loading and simple exploratory visualizations, and VTK and Paraview libraries for more complex visualizations. Blue Waters has support for yt and Paraview as well.

yt has built in support for basic volume rendering and it is possible to produce videos by generating a sequence of plots.

Visualization Design

I will be working first on a visualization method that identifies and shows the evolution of the halos over time. Subsequently, I will work on an interactive tool that allows users to zoom into individual halos and compare their structure.
